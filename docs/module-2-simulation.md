## Learning Outcomes
- Simulate physics, gravity, and collisions in **Gazebo**.
- Build environments and humanoid models in **Unity**.
- Simulate **LiDAR, Depth Cameras, IMUs** for robot perception.

## Module Overview
The Digital Twin module helps bridge the gap between **virtual simulation** and **real-world robots**. You will:

1. Set up Gazebo simulation environment.
2. Create and test URDF/SDF robots.
3. Simulate sensors and test interactions in Unity.
4. Understand **physics-based simulation** for humanoids.

## Key Tools
- **Gazebo Simulator**
- **Unity 3D**
- **URDF/SDF robot formats**
- **Python / ROS 2 integration**
