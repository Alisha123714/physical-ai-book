## Project Overview
In this capstone, students integrate **all previous modules** to build an autonomous humanoid robot that:

- Receives **voice commands**.
- Plans a path around **obstacles**.
- Recognizes and **manipulates objects**.
- Demonstrates **natural human-robot interaction**.

## Requirements
- ROS 2 navigation and action pipelines.
- Gazebo & Unity simulation.
- NVIDIA Isaac perception and control.
- VLA integration for voice and vision.

## Deliverables
1. A **simulated humanoid robot demo**.
2. Code deployed on **Jetson Orin or cloud simulator**.
3. Presentation video (≤90 seconds).
