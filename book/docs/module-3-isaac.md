## Learning Outcomes
- Understand **NVIDIA Isaac Sim** for photorealistic robot simulation.
- Use **Isaac ROS** for navigation and perception.
- Train reinforcement learning pipelines for robot control.

## Module Overview
The AI-Robot Brain module focuses on advanced perception and motion:

1. Generate synthetic data with Isaac Sim.
2. Plan paths using **Nav2** for bipedal humanoids.
3. Deploy perception pipelines on **Edge devices** (Jetson Orin).

## Key Tools
- **NVIDIA Isaac Sim**
- **Isaac ROS SDK**
- **Nav2 Path Planning**
- **Python / ROS 2 integration**
